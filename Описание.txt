CookBook — Личная книга рецептов
Простой сайт на Flask, где можно хранить свои рецепты и смотреть чужие. У каждого блюда есть название и фотография

Функционал:
Авторизация: вход и регистрация пользователей
Рецепты: создание записей с названием и загрузкой фото блюда
База данных (ORM): хранение юзеров и данных в SQLite через SQLAlchemy
Интерфейс: адаптивный дизайн на Bootstrap 5
REST API: выдача списка рецептов в формате JSON по адресу /api

Стек:
Бэкенд: Python, Flask
Фронтенд: Jinja2, Bootstrap
Хранение: SQLite + папка для файлов

Запуск:
1 pip install -r requirements.txt
2 python app.py
3 Адрес: http://127.0.0.1:5000
