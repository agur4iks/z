PlantSwap — Платформа для обмена комнатными растениями
Данный учебный проект представляет собой веб-приложение на базе Flask, предназначенное для публикации объявлений о передаче растений в добрые руки и их бронирования.

Функционал:
Интерфейс на Bootstrap
Загрузка и хранение файлов
Постоянное хранение данных
Система авторизации
Механика бронирования
Динамические шаблоны

Стек технологий
Язык программирования: Python 3.x.
Фреймворк: Flask.
База данных: SQLite (через Flask-SQLAlchemy).
Фронтенд: HTML, CSS, Bootstrap 5.
Хранение данных: SQL (база данных), статические файлы (изображения).

Инструкция по запуску:
Установите - pip install flask flask-sqlalchemy
Запустите сервер приложения - python app.py
Перейдите по адресу - http://127.0.0.1:5000
